# Security Policy

## Supported Versions

Older versions of amen Share do not receive security updates. To ensure your system remains secure, we strongly recommend updating amen Share X regularly. You can automate these updates using tools like [Watchtower](https://github.com/containrrr/watchtower).

## Reporting a Vulnerability

Thank you for taking the time to report a vulnerability. Please DO NOT create an issue on GitHub because the vulnerability could get exploited. Instead please write an email to [me@amen-edu.me](mailto:me@amen-edu.me).
